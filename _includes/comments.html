{%- comment -%} giscus comments, backed by this repo's GitHub Discussions. Included by the post layout only when a post sets "comments: true" in its front matter. Prerequisite: the giscus GitHub App must be installed on this repository (https://github.com/apps/giscus). Until then the widget shows a configuration message instead of a comment box. The script is injected dynamically so we can set the correct initial theme (matching the site's light/dark state) before the giscus iframe loads, avoiding a flash. theme.js keeps it in sync afterwards when the user toggles the theme. Comments are stored in this repo's "Comments" discussion category. To change categories, update data-category / data-category-id below. {%- endcomment -%}
